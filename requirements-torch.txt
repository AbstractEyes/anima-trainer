# torch is NOT `pip install -r`-able from here — it needs an explicit CUDA wheel index.
# torch wheels bundle their own CUDA runtime, so your local nvcc/toolkit version is irrelevant.
#
# LOCAL smoke-test box (RTX 4090, sm_89, Windows, py3.12) — use cu128 for parity with
# the Blackwell sm_120 target (cu128 wheels also run fine on the 4090):
#
#     pip install torch torchvision --index-url https://download.pytorch.org/whl/cu128
#
# If the cu128 win/py3.12 wheel is unavailable at install time, fall back to cu126:
#
#     pip install torch torchvision --index-url https://download.pytorch.org/whl/cu126
#
# TARGET training box (RTX PRO 6000 Blackwell, sm_120, Linux) — Blackwell REQUIRES
# torch >= 2.7 / cu128. torch + deepspeed there come from diffusion-pipe's own
# requirements.txt inside the conda env:
#
#     pip install -r external/diffusion-pipe/requirements.txt
#
# Do NOT install flash-attn anywhere — it is broken on sm_120; diffusion-pipe uses SDPA.
